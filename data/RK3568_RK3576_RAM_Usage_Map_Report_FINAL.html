3) 산출 기준

  • Kernel used: dumpsys meminfo 요약의 Used RAM = (used pss + kernel)에서 kernel 값.
  • Android OS(userspace, used pss): 동일 요약의 used pss 값 (프로세스/서비스 전체의 PSS 합에서 캐시 제외된 “사용중” 관점).
  • Reclaimable cache: cached pss + cached kernel (회수 가능한 페이지 캐시/커널 캐시 관점).
  • ION: Android 12.1은 요약에 ION으로 표기. Android 15는 DMA-BUF로 표기되어 ION 대신 DMA-BUF 값으로 대응.
  • Product Apps PSS: Total PSS by process에서 패키지 prefix 기준 합산 (com.wallpad.*, com.kdiwin.*). 이 값은 캐시 포함일 수 있음.
